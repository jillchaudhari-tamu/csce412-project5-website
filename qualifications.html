Qualifications

Skills & Areas of Interest

I’ve developed a broad set of technical skills through both coursework and hands-on projects. Programming languages like Python, C++, Java, and JavaScript are tools I use to turn ideas into real applications, whether it’s building interactive web apps, automating workflows, or analyzing complex datasets. With HTML and CSS, I create clean, user-friendly interfaces that make my applications accessible and visually engaging.

My experience with frameworks and libraries such as React, Node.js, Express, Flask, TensorFlow, PyTorch, and Scikit-learn allows me to take projects from concept to completion. Tools like Pandas, NumPy, Matplotlib, and Apache Spark let me explore and analyze large datasets, uncover patterns, and generate insights that guide decision-making.

Working with cloud platforms like AWS and Microsoft Azure, and containerization tools such as Docker and Kubernetes, has given me the ability to deploy scalable solutions efficiently. I also work with databases including MySQL, MongoDB, and Firestore, ensuring that applications can manage real-world data reliably and securely.

I’m particularly interested in machine learning, AI-powered web applications, and data-driven solutions for environmental sustainability. Machine learning fascinates me because it combines creativity with problem-solving, allowing me to design systems that learn from data and make intelligent predictions. AI-powered web applications let me connect these models to real users in meaningful ways, creating tools that are both functional and impactful. I’m drawn to environmental applications because I want my technical work to contribute to addressing pressing real-world problems, whether through predictive models, research, or innovative software solutions.

Beyond technical skills, I strive to apply my knowledge to projects that make a tangible difference, combining curiosity, creativity, and technical rigor to build solutions that have a real-world impact.
